Rust 每周深度学习计划

调整日期：2026 年 3 月 12 日
计划类型：周计划（每周一篇深度内容）
学习节奏：12 周完成核心内容

📋 计划说明

原日计划（112 天）调整为周计划（12 周），每周一个主题，内容更聚焦、更深入。

每周结构：

  • 1 篇核心文章（2000-3000 字）
  • 3-5 个代码示例
  • 2-3 个练习题
  • 推荐资源

📅 12 周学习大纲

第 1 周：Rust 入门与环境搭建

主题：为什么选择 Rust + 第一个程序

内容：

  • Rust 的核心特性（内存安全、零成本抽象、并发安全）
  • 安装 rustup 和 cargo
  • Hello World 和 cargo 项目结构
  • 变量、可变性、基本数据类型

代码示例：

  • cargo new 项目
  • 变量声明与模式匹配
  • 基本类型操作

练习：

  1. 安装 Rust 并运行第一个程序
  2. 创建一个计算器（加减乘除）
  3. 实现温度转换（摄氏度↔华氏度）

第 2 周：所有权与借用（上）

主题：理解 Rust 的核心——所有权系统

内容：

  • 所有权三规则
  • 移动语义（Move）
  • Clone 与 Copy trait
  • 作用域与 drop

代码示例：

  • 所有权转移示例
  • clone() vs 移动
  • Copy 类型演示

练习：

  1. 修复所有编译错误（提供错误代码）
  2. 实现字符串处理函数
  3. 理解何时需要 clone()

第 3 周：所有权与借用（下）

主题：引用、切片与借用规则

内容：

  • 不可变引用与可变引用
  • 借用规则（多读一写）
  • 字符串切片
  • 悬垂引用

代码示例：

  • 借用与引用
  • 字符串切片操作
  • 函数参数中的切片

练习：

  1. 实现字符串单词提取器
  2. 修复借用冲突
  3. 切片操作实践

第 4 周：结构体与枚举

主题：自定义数据类型

内容：

  • 结构体定义与实例化
  • 方法实现（impl）
  • 枚举与模式匹配
  • Option 和 Result 简介

代码示例：

  • 结构体与方法
  • match 表达式
  • Option 使用场景

练习：

  1. 创建学生管理系统
  2. 实现简单的枚举状态机
  3. 使用 Option 处理空值

第 5 周：错误处理

主题：优雅的 panic 与 Result

内容：

  • panic! 宏
  • Result<T, E> 类型
  • ? 操作符
  • 自定义错误类型

代码示例：

  • 文件读取错误处理
  • 自定义错误枚举
  • 错误传播链

练习：

  1. 实现带错误处理的配置文件读取
  2. 创建自定义错误类型
  3. 使用 ? 简化错误处理

第 6 周：泛型与 Trait

主题：抽象与复用

内容：

  • 泛型函数与结构体
  • Trait 定义与实现
  • Trait Bound
  • 常用标准库 Trait

代码示例：

  • 泛型数据结构
  • Trait 实现
  • where 子句

练习：

  1. 实现泛型栈
  2. 为自定义类型实现 Display Trait
  3. 使用 Trait Bound 约束泛型

第 7 周：生命周期

主题：理解引用的有效期

内容：

  • 生命周期概念
  • 生命周期标注
  • 省略规则
  • 结构体中的生命周期

代码示例：

  • 生命周期标注示例
  • 函数返回引用
  • 结构体生命周期

练习：

  1. 修复生命周期错误
  2. 实现带生命周期的结构体
  3. 理解生命周期省略规则

第 8 周：集合类型

主题：Vec、String、HashMap

内容：

  • Vec 动态数组
  • String 字符串
  • HashMap 哈希表
  • 集合的性能考虑

代码示例：

  • Vec 增删改查
  • 字符串拼接
  • HashMap 统计词频

练习：

  1. 实现待办事项列表
  2. 词频统计工具
  3. 集合转换与迭代

第 9 周：闭包与迭代器

主题：函数式编程特性

内容：

  • 闭包语法与捕获
  • Iterator trait
  • 适配器（map、filter、fold）
  • 惰性求值

代码示例：

  • 闭包捕获环境
  • 迭代器链式调用
  • collect() 转换

练习：

  1. 实现自定义迭代器
  2. 使用迭代器处理数据
  3. 闭包排序与过滤

第 10 周：智能指针

主题：Box、Rc、RefCell、Arc、Mutex

内容：

  • Box 堆分配
  • Rc 引用计数
  • RefCell 内部可变性
  • Arc + Mutex 线程安全

代码示例：

  • 递归类型用 Box
  • Rc 多所有权
  • Arc + Mutex 并发

练习：

  1. 实现树形数据结构
  2. 多所有者场景
  3. 线程安全计数器

第 11 周：并发编程

主题：线程、通道、async/await

内容：

  • 创建线程
  • 消息传递（mpsc）
  • async/await 基础
  • Tokio 简介

代码示例：

  • 多线程计算
  • 通道通信
  • 异步 HTTP 请求

练习：

  1. 并行计算数组和
  2. 生产者 - 消费者模型
  3. 异步文件读取

第 12 周：实战项目

主题：综合应用

项目选项：

  1. 命令行工具：文件搜索（类似 grep）
  2. Web 服务：使用 Axum 构建 REST API
  3. 系统工具：简单的任务管理器

内容：

  • 项目规划
  • 代码组织
  • 测试与文档
  • 打包发布
