数据库内核工程师学习路线：从 C++ 到分布式系统

2026-03-05
2026-03-05

📌 路线背景

基于 OceanBase 日志系统/高可用架构研发岗位 要求制定，目标是从通信领域 C++ 开发转型到数据库内核研发。

目标岗位核心要求 

1
2
3
4
5

1. 熟练掌握 C/C++ 语言，熟悉数据结构、算法、操作系统
2. 熟悉关系数据库和/或分布式系统的运行机制和架构体系
3. 加分项：一致性协议、分布式事务、存储引擎经验
4. 加分项：底层系统研发经验或关系数据库研发经验
5. 加分项：熟悉 Rust 语言，或有意愿使用 Rust 开发分布式系统

个人背景分析

维度 当前状态 目标状态 差距
C++ 能力 2 年半基站数据解析经验 数据库内核级 C++ ⚠️ 中等
数据库知识 应用层使用 内核研发 ⚠️ 较大
分布式系统 有限接触 Paxos、一致性协议 ⚠️ 较大
Rust 语言 0 基础 能用于系统开发 ⚠️ 从零开始
存储引擎 未涉及 WAL、日志系统 ⚠️ 较大

预计学习周期：6-12 个月（在职学习）

📚 第一阶段：C++ 深度提升（第 1-2 月）

目标

达到数据库内核研发要求的 C++ 水平，超越应用层开发。

学习内容

1.1 C++ 高级特性

主题 具体内容 资源推荐 耗时
内存管理 智能指针、自定义分配器、内存池 《Effective C++》第 1-3 章 1 周
模板编程 模板元编程、SFINAE、constexpr 《C++ Templates》前 8 章 2 周
移动语义 右值引用、完美转发、移动优化 《Effective Modern C++》 1 周
并发编程 线程、锁、原子操作、无锁数据结构 《C++ Concurrency in Action》 2 周

1.2 数据结构与算法强化

主题 具体内容 练习平台 目标
高级数据结构 B/B+ 树、跳表、哈希表、布隆过滤器 LeetCode 50 题
字符串算法 KMP、Trie、后缀数组 LeetCode 20 题
图算法 最短路径、拓扑排序、并查集 LeetCode 30 题

1.3 操作系统深入

主题 具体内容 资源推荐
进程/线程 调度、上下文切换、线程池 《操作系统导论》
内存管理 虚拟内存、页表、TLB 《深入理解计算机系统》
I/O 系统 缓冲、异步 I/O、mmap 《Linux/UNIX 系统编程手册》
文件系统 日志文件系统、写时复制 阅读 ext4 文档

实践项目

  1. 实现一个内存池 - 支持定长/变长对象分配
  2. 实现一个线程池 - 支持任务提交、优雅关闭
  3. 实现一个 B+ 树 - 支持插入、删除、范围查询

🌐 第三阶段：分布式系统（第 5-6 月）

目标

3.1 分布式系统基础

主题 具体内容 资源推荐

时间与时钟 逻辑时钟、向量时钟、混合逻辑时钟 论文阅读

复制技术 主从复制、多主复制、无主复制 《DDIA》Ch5

🛠️ 第五阶段：实战项目（第 9-10 月）

目标

通过实战项目整合所学知识，形成完整知识体系。

推荐项目

项目 1：用 Rust 实现一个简单的 KV 存储引擎

技术栈：Rust + 存储引擎 + WAL

功能要求：

  • 支持 PUT/GET/DELETE 操作
  • 基于 LSM-Tree 或 B+ 树
  • 实现 WAL 日志
  • 支持崩溃恢复

参考项目：

  • Sled - Rust 实现的嵌入式数据库
  • RocksDB - Facebook 的 KV 存储

项目 2：用 Rust 实现 Raft 协议

技术栈：Rust + 分布式系统 + Raft

功能要求：

  • Leader 选举
  • 日志复制
  • 成员变更
  • 持久化

参考项目：

  • tikv/raft-rs - TiKV 的 Raft 实现
  • async-raft - 异步 Raft

项目 3：阅读并贡献开源项目

推荐项目：

项目 语言 方向 难度
TiKV Rust 分布式 KV 存储 ⭐⭐⭐⭐⭐
RisingWave Rust 流式数据库 ⭐⭐⭐⭐⭐
