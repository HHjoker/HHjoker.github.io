1. 不可变引用

这是 Rust 学习的重要部分。让我们深入了解：

核心概念

Rust 的设计理念是零成本抽象和内存安全。这意味着：

  • 不需要垃圾回收器
  • 编译期检查内存安全
  • 高性能，与 C/C++ 相当

// 借用示例
fn main() {
    let s = String::from("hello");
    
    // 不可变借用
    let r1 = &s;
    let r2 = &s;
    
    // 可变借用（同一时刻只能有一个）
    let r3 = &mut s;
    
    println!("{} {} {}", r1, r2, r3);
}

  1. 理解概念：确保理解基本原理
  2. 动手实践：运行代码，尝试修改
  3. debug 经验：遇到错误时仔细阅读编译器提示

3. 常见问题与解答

Q1: 为什么 Rust 的所有权系统这么设计？

A: 主要是为了在没有垃圾回收的情况下保证内存安全。

Q2: 遇到编译错误怎么办？

A: Rust 的编译器错误信息非常友好，仔细阅读并按建议修改。

Q3: 如何记住这么多规则？

A: 多写代码，形成肌肉记忆。规则其实很一致。
