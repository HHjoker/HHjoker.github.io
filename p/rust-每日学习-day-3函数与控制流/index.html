关键点：

  • 参数必须标注类型
  • 返回值用 -> 指定
  • 最后一个表达式无分号 = 返回值

fn main() {
    // 语句（Statement）：执行操作，不返回值
    let x = 5;  // let 绑定是语句
    
    // 表达式（Expression）：计算并返回值
    let y = {   // 块是表达式
        let a = 3;
        let b = 4;
        a + b   // 没有分号，返回 7
    };
    
    println!("x = {}, y = {}", x, y);
    
    // if 也是表达式
    let num = 10;
    let description = if num > 5 { "big" } else { "small" };
    println!("{}", description);
}

区别：

语句 表达式
不返回值 返回值
以分号结束 不以分号结束
let x = 5; 5
if cond { } (有分号) if cond { a } else { b }

fn calculate(x: i32, y: i32) -> (i32, i32, i32) {
    let sum = x + y;
    let diff = x - y;
    let product = x * y;
    (sum, diff, product)

}

fn main() {
    let mut counter = 0;
    
    // 无限循环，需要手动 break
    let result = loop {
        counter += 1;
        
        if counter == 10 {
            break counter * 2;  // break 可以返回值
        }
    };
    
    println!("Counter reached {}, result = {}", counter, result);
}

fn main() {
    let mut count = 0;
    
    'counting_up: loop {
        println!("count = {}", count);
        
        let mut remaining = 10;
        
        loop {
            println!("remaining = {}", remaining);
            
            if remaining == 9 {
                break;  // 只跳出内层循环
            }
            
            if count == 2 {
                break 'counting_up;  // 跳出外层循环
            }
            
            remaining -= 1;
        }
        
        count += 1;
    }
    
    println!("End count = {}", count);
}

fn main() {
    // 遍历数组
    let arr = [10, 20, 30, 40, 50];
    
    for element in arr {
        println!("Value: {}", element);
    }
    
    // 使用 enumerate 获取索引
    for (index, value) in arr.iter().enumerate() {
        println!("arr[{}] = {}", index, value);
    }
    
    // 范围迭代
    for i in 1..4 {
        println!("i = {}", i);  // 1, 2, 3（不包含 4）
    }
    
    // 包含上限的范围
    for i in 1..=4 {
        println!("i = {}", i);  // 1, 2, 3, 4
    }
    
    // 倒计时
    for i in (1..=5).rev() {
        println!("{}!", i);
    }
}

fn main() {
    let arr = [10, 20, 30, 40, 50];
    
    // while 方式（容易出错）
    let mut index = 0;
    while index < arr.len() {
        println!("while: {}", arr[index]);
        index += 1;
    }
    
    // for 方式（更安全、简洁）
    for element in &arr {
        println!("for: {}", element);
    }
    
    // for 自动处理边界，不会越界
}

fn main() {
    let some_value: Option<i32> = Some(5);
    
    // match 写法
    match some_value {
        Some(x) => println!("Got: {}", x),
        _ => (),  // 其他情况什么都不做
    }
    
    // if let 写法（更简洁）
    if let Some(x) = some_value {
        println!("Got: {}", x);
    }
    
    // if let else
    let value: Option<i32> = None;
    if let Some(x) = value {
        println!("Got: {}", x);
    } else {
        println!("Got nothing");
    }
}

fn main() {
    let arr = [1, 2, 3, 4, 5];
    
    // 借用迭代（最常用）
    for x in &arr {
        println!("{}", x);  // x 是 &i32
    }
    
    // 可变借用迭代
    let mut arr2 = [1, 2, 3];
    for x in &mut arr2 {
        *x += 1;  // x 是 &mut i32，需要解引用
    }
    println!("{:?}", arr2);  // [2, 3, 4]
    
    // 所有权迭代（消耗数组）
    for x in arr {
        println!("{}", x);  // x 是 i32
    }
    // arr 之后不能再使用
}

// 任务：用 if let 处理 Option 和 Result

fn find_element(arr: &[i32], target: i32) -> Option<usize> {
    for (i, &val) in arr.iter().enumerate() {
        if val == target {
            return Some(i);
        }
    }
    None
}

fn main() {
    let arr = [1, 5, 10, 15, 20];
    
    // 用 if let 处理查找结果
    if let Some(index) = find_element(&arr, 10) {
        println!("Found 10 at index {}", index);
    } else {
        println!("10 not found");
    }
}

💭 今日感悟

函数是 Rust 的基石之一。记住：最后一个表达式无分号就是返回值，这个规则一开始容易忘记。

if/else 在 Rust 中是表达式而不是语句，这个设计很优雅。可以直接赋值，代码更简洁。

for 循环比 while 更安全，因为它自动处理边界，不会越界。以后应该优先使用 for。

match 是 Rust 最强大的特性之一。它强制你处理所有情况，避免了遗漏分支的 bug。配合枚举使用效果更佳。

if let 和 while let 是 match 的语法糖，在只关心一个分支时很方便。

今天的练习要全部动手写一遍，特别是 FizzBuzz 和 match 练习，能帮助巩固理解。
