1. 什么是 Slice？

Slice 是 Rust 中一种引用类型，它允许你引用集合中连续的元素序列，而不需要复制或拥有这些数据。

核心特点

  • 不拥有数据：Slice 只是对已有数据的引用
  • 固定大小：Slice 的大小在编译时确定（胖指针：包含指针 + 长度）
  • 安全访问：编译期检查边界，避免越界

Slice vs 数组

特性 数组 Slice
类型 [T; N]（固定大小） &[T]（动态大小）
存储 栈上 栈上（引用）+ 堆上（数据）
大小 编译时已知 运行时确定
所有权 拥有数据 借用数据

fn main() {
    // 定义一个数组
    let arr = [1, 2, 3, 4, 5];
    
    // 创建切片 - 引用整个数组
    let slice1 = &arr[..];
    println!("整个数组：{:?}", slice1);  // [1, 2, 3, 4, 5]
    
    // 创建切片 - 引用部分元素
    let slice2 = &arr[1..4];  // 从索引 1 到 3（不包括 4）
    println!("部分切片：{:?}", slice2);  // [2, 3, 4]
    
    // 省略写法
    let slice3 = &arr[0..2];  // [1, 2]
    let slice4 = &arr[..3];   // [1, 2, 3]
    let slice5 = &arr[2..];   // [3, 4, 5]
}

fn main() {
    let arr = [10, 20, 30, 40, 50];
    
    // a..b  : 从 a 到 b-1
    let s1 = &arr[1..3];   // [20, 30]
    
    // a..=b : 从 a 到 b（包含 b）
    let s2 = &arr[1..=3];  // [20, 30, 40]
    
    // ..b   : 从 0 到 b-1
    let s3 = &arr[..3];    // [10, 20, 30]
    
    // a..   : 从 a 到末尾
    let s4 = &arr[2..];    // [30, 40, 50]
    
    // ..    : 全部
    let s5 = &arr[..];     // [10, 20, 30, 40, 50]
}

// 计算切片的平均值
fn average(numbers: &[i32]) -> f64 {
    if numbers.is_empty() {
        return 0.0;
    }
    let sum: i32 = numbers.iter().sum();
    sum as f64 / numbers.len() as f64
}

// 查找最大值
fn find_max(numbers: &[i32]) -> Option<i32> {
    numbers.iter().max().copied()
}

// 打印切片内容
fn print_slice(label: &str, data: &[i32]) {
    println!("{}: {:?}", label, data);
}

fn main() {
    let scores = [92, 85, 78, 90, 88];
    
    // 传递整个数组的切片
    let avg = average(&scores);
    println!("平均分：{:.2}", avg);  // 平均分：86.60
    
    // 传递部分切片
    let top_three = &scores[0..3];
    let top_avg = average(top_three);
    println!("前三门平均分：{:.2}", top_avg);  // 前三门平均分：85.00
    
    // 查找最高分
    if let Some(max) = find_max(&scores) {
        println!("最高分：{}", max);  // 最高分：92
    }
    
    // 使用打印函数
    print_slice("所有成绩", &scores);
    print_slice("前三门", top_three);
}

fn main() {
    // String - 拥有的、可增长的字符串（堆上）
    let mut owned = String::from("hello");
    owned.push_str(" world");
    
    // &str - 字符串切片（引用）
    let slice: &str = &owned[0..5];  // "hello"
    
    // 字符串字面量本身就是 &str
    let literal: &str = "I am a &str";
    
    println!("owned: {}", owned);
    println!("slice: {}", slice);
    println!("literal: {}", literal);
}

// 返回字符串的第一个单词
fn first_word(s: &str) -> &str {
    let bytes = s.as_bytes();
    
    for (i, &item) in bytes.iter().enumerate() {
        if item == b' ' {
            return &s[0..i];
        }
    }
    
    &s[..]  // 如果没有空格，返回整个字符串
}

// 检查是否包含某子串
fn contains_substring(haystack: &str, needle: &str) -> bool {
    haystack.contains(needle)
}

// 转换为大写
fn to_uppercase(s: &str) -> String {
    s.to_uppercase()
}

fn main() {
    let greeting = String::from("Hello Rust!");
    
    // 获取第一个单词
    let first = first_word(&greeting);
    println!("第一个单词：{}", first);  // Hello
    
    // 检查包含
    let has_rust = contains_substring(&greeting, "Rust");
    println!("包含 Rust: {}", has_rust);  // true
    
    // 转换大写
    let upper = to_uppercase(&greeting);
    println!("大写：{}", upper);  // HELLO RUST!
    
    // 字符串字面量也可以直接使用
    let literal_word = first_word("Good morning");
    println!("字面量的第一个单词：{}", literal_word);  // Good
}

fn main() {
    let mut numbers = [1, 2, 3, 4, 5];
    
    // 可变切片 - 可以修改元素
    let slice = &mut numbers[1..4];
    
    // 修改切片中的元素
    for num in slice.iter_mut() {
        *num *= 2;
    }
    
    println!("修改后的数组：{:?}", numbers);  // [1, 4, 6, 8, 5]
}

// 使用可变切片的函数
fn double_all(numbers: &mut [i32]) {
    for num in numbers.iter_mut() {
        *num *= 2;
    }
}

fn main() {
    let mut data = [10, 20, 30, 40, 50];
    double_all(&mut data);
    println!("翻倍后：{:?}", data);  // [20, 40, 60, 80, 100]
}

fn main() {
    let arr = [1, 2, 3, 4, 5];
    let slice = &arr[..];
    
    // len() - 获取长度
    println!("长度：{}", slice.len());  // 5
    
    // is_empty() - 检查是否为空
    println!("是否为空：{}", slice.is_empty());  // false
    
    // iter() - 迭代
    print!("迭代：");
    for item in slice.iter() {
        print!("{} ", item);
    }
    println!();  // 迭代：1 2 3 4 5
    
    // iter_mut() - 可变迭代
    let mut arr2 = [1, 2, 3];
    for num in arr2.iter_mut() {
        *num += 10;
    }
    println!("加 10 后：{:?}", arr2);  // [11, 12, 13]
    
    // split_at() - 在指定位置分割
    let (left, right) = slice.split_at(2);
    println!("左边：{:?}", left);   // [1, 2]
    println!("右边：{:?}", right);  // [3, 4, 5]
    
    // windows() - 滑动窗口
    print!("滑动窗口 (大小 2): ");
    for window in slice.windows(2) {
        print!("{:?} ", window);
    }
    println!();  // [1, 2] [2, 3] [3, 4] [4, 5]
    
    // chunks() - 分块
    print!("分块 (大小 2): ");
    for chunk in slice.chunks(2) {
        print!("{:?} ", chunk);
    }
    println!();  // [1, 2] [3, 4] [5]
}

🔑 关键概念理解

1. Slice 是"胖指针"

Slice 在内存中包含两个部分：

  • 指针：指向数据的起始位置
  • 长度：Slice 包含多少个元素

这使得 Slice 的大小在运行时才能确定，但访问是安全的。

2. Slice 不拥有数据

}  // arr 和 slice 都在这里失效

💪 练习时间

基础练习

  1. 切片求和：编写一个函数 sum_slice，接收 &[i32] 参数，返回所有元素的和。

  2. 反转切片：编写一个函数 reverse_in_place，接收 &mut [i32] 参数，原地反转切片中的元素顺序。

  3. 查找索引：编写一个函数 find_index，接收 &[i32] 和要查找的值，返回该值在切片中的索引（使用 Option<usize>）。

  4. 字符串处理：编写一个函数 last_word，接收 &str 参数，返回最后一个单词（假设单词由空格分隔）。

  5. 统计字符：编写一个函数 count_char，接收 &str 和一个 char，返回该字符在字符串中出现的次数。

进阶挑战

  1. 实现二分查找：对已排序的切片实现二分查找算法。

  2. 滑动窗口最大值：给定一个数组和窗口大小 k，找出每个滑动窗口中的最大值。

  3. 字符串压缩：实现简单的游程编码（RLE），如 “aaabbc” → “a3b2c1”。

实战项目

  1. 文本分析工具：读取文本文件，统计词频、行数、字符数。

  2. 命令行计算器：支持基本的算术运算，使用切片处理输入。

  3. 简单 grep：在文件中搜索包含特定模式的行。

// 1. 切片求和
fn sum_slice(numbers: &[i32]) -> i32 {
    numbers.iter().sum()
}

// 2. 反转切片
fn reverse_in_place(slice: &mut [i32]) {
    let mut left = 0;
    let mut right = slice.len() - 1;
    
    while left < right {
        slice.swap(left, right);
        left += 1;
        if right == 0 { break; }
        right -= 1;
    }
}

// 3. 查找索引
fn find_index(slice: &[i32], target: i32) -> Option<usize> {
    for (i, &val) in slice.iter().enumerate() {
        if val == target {
            return Some(i);
        }
    }
    None
}

// 4. 最后一个单词
fn last_word(s: &str) -> &str {
    s.trim()
        .rsplit_whitespace()
        .next()
        .unwrap_or("")
}

// 5. 统计字符
fn count_char(s: &str, c: char) -> usize {
    s.chars().filter(|&ch| ch == c).count()
}

Slice 是 Rust 中非常优雅的设计。它让我们能够：

  1. 高效地传递数据：不需要复制整个数组或字符串
  2. 安全地访问数据：编译期检查边界，避免越界错误

理解 Slice 是掌握 Rust 所有权系统的重要一步。它展示了 Rust 如何在零成本抽象的前提下，提供内存安全保障。
